This is a generic application framework to create web-based or/and desktop applications with the same business logic. The model and the view layer fully detached from the presenter logic. You can use various library both for create user interface (PHP-GTK, Smarty, Dwoo, etc) and for presist data (Zend\_Db, AdoDB, Doctrine, etc). Other characteristic: strong usage of interfaces and design patterns; well documented source code; reusable libraries.
